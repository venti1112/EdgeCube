## 二、服务说明

本应用提供以下主要功能：

1. **服务端管理**：在 Android 设备上创建、配置、启动和管理 Minecraft 服务器实例，支持原版（Vanilla）、PaperMC、Forge、Fabric、NeoForge、Velocity、PowerNukkitX、PocketMine 等多种服务端。首次启动服务端时，本应用会弹出 Minecraft EULA 确认框，点击‘同意并启动’后方可运行服务端；若您不同意，请立即卸载本软件。
2. **文件管理**：浏览、编辑、导入导出实例目录下的文件。
3. **网络服务**：提供 FTP、SSH（SFTP 与终端）、MCP（模型上下文协议）等网络服务，以及 UPnP 端口映射与 FRP 隧道功能。
4. **Shell 终端**：在设备上执行 shell 命令，支持交互式终端与单次命令执行两种模式。
5. **模组/插件管理**：通过 Modrinth 等第三方平台搜索、下载和管理模组与插件，支持哈希校验。
6. **运行环境管理**：支持以 .ecpkg 包形式导入和管理 Java、PHP、frpc 等运行环境，并可检查更新。
7. **在线服务**：可选启用，用于崩溃日志上传，以改进软件体验。
8. **应用更新**：当检测到新版本时，本应用可提示更新 。
9. **文件关联**：本应用注册了 .ecpkg 文件扩展名的关联，可从外部文件管理器直接打开导入运行环境包。

## 四、隐私与数据收集

1. **设备标识**：当您启用"在线服务"时，本应用会生成一个设备唯一标识用于服务识别。生成优先级为：OAID → AAID → GUID。标识前缀 AA 表示 OAID/AAID、GG 表示 GUID。该标识存储于您的设备本地配置文件中，在崩溃日志上传时作为请求头发送。
2. **崩溃日志**：当您启用在线服务且服务端或隧道异常退出时，您可选择上传崩溃日志以帮助开发者改进软件。上传的信息包括：
   - 服务端控制台日志
   - 设备信息：SoC 型号、架构、制造商、设备型号、Android 版本、安全补丁日期
   - 系统信息：总内存/已用内存、CPU 使用率
   - 服务端运行类型与运行时 ID、退出码
   - 设备标识
3. **本地存储**：您的实例文件、配置、运行环境等均存储在设备本地（默认为外部存储的 EdgeCube 目录，可在设置中自定义），开发者不会主动收集这些数据。
4. **系统权限申请**：
   - **文件管理权限**：本应用需要 MANAGE_EXTERNAL_STORAGE 权限以读写服务端实例文件。
   - **通知权限**：用于在后台运行服务端时发送持久通知。
   - **本地网络权限**：用于发现和显示设备局域网 IP 地址。
   - **电池优化豁免**：本应用可请求关闭电池优化以提升服务端后台运行的稳定性。
5. **QQ 群跳转**：首次启动时本应用会提供官方 QQ 群跳转链接，该操作会打开外部 QQ 应用。
6. **系统资源监测**：本应用会采集设备的 CPU 与内存使用情况，仅用于本地实时显示，不上传至服务器。
7. **第三方服务**：
   - 下载服务端或模组时，本应用会向官方源或 MSL 镜像源（mslmc.cn）、Modrinth 等第三方服务发起网络请求，这些服务有其独立的隐私政策。
   - 使用 FRP 隧道时，您的流量将经过您配置的 frps 服务器。
8. **不收集个人信息**：除上述情况外，本应用不会收集您的个人身份信息。您可随时在应用内"设置 → 在线服务"中关闭在线服务（关闭后崩溃日志上传和设备标识相关功能将停止）。
